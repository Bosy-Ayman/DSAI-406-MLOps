## Implementation 

The assignment has three phases: 

**Phase 1**:  Student A creates a ML script that uses any library version they want to train a generative AI model (a simple GAN) to generate images. Include no documentation or requirements.txt file. But Student A needs to send the notebook and the data file (csv file) to student B. 

**Phase 2**: Student B must try to run Student A’s notebook on their own machine and try to get the exact same accuracy score.

**Phase 3**: Student B must write a report (1 page) answering:
- How many commands did I have to run before it worked?
- What libraries were missing? Did version mismatches cause errors?
- Did the model produce the same result? If not, why? (Random seeds, data shuffling, etc.)
- If this had to run on a server at 3:00 AM, would it survive?

You need to upload Student A's code and your report.
